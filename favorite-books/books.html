The C Programming Language- Dennis Ritchie

This book is meant to help the reader learn how to program in C. It is the definitive reference guide, now in a second edition. This second edition brings the classic original up to date to include the ANSI standard.

What does it contain?

It is basically a reference book for C 1989 standard, also called ANSI-C.

My thoughts about it

Starting with the basic "Hello World" program, this book covers everything (of course, as the official guide to the language this is expected). It's been useful for me as a beginner.
